## Admin Guide

### Login
- Go to `/auth/login`
- Sign in with admin email (set in env `ADMIN_EMAIL`)
- If new device, you’ll be asked to enter OTP sent to admin email.

### Edit website text
- `/admin/content`
- Update Home/About/Services/Global settings
- Click **Save changes**

### Review applications
- `/admin/applications`
- Click an application → view details + uploads
- Set status (in_review/approved/rejected)
- Add internal note
- Optional: write message to applicant (will email applicant)

### Send bulk messages
- `/admin/messages`
- Select audience
- Write subject + message
- Send
